🌍 REST Country – Interfaccia di esplorazione dei Paesi del mondo
Un’applicazione web moderna che permette di esplorare i Paesi del mondo tramite l’API REST Countries.

✨ Funzionalità principali


🔎 Ricerca dei Paesi
- Barra di ricerca per filtrare i Paesi per nome.
- Risultati aggiornati in tempo reale mentre l’utente digita.
  
🌐 Filtri per Regione
- Menu a tendina per selezionare una regione (Africa, Asia, Europe, Oceania, Americas, ecc.).
- Filtraggio immediato della lista dei Paesi.
  
🏳️ Card dei Paesi
Ogni Paese viene mostrato tramite una card che include:
- Bandiera nazionale
- Nome del Paese
- Population
- Region
- Capital
  
🌗 Modalità Light/Dark
- Switch in alto a destra per cambiare tema.
- Persistenza del tema tramite localStorage (se implementato).
